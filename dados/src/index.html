🚀 Primeiros Passos: Guia Rápido

Se você é novo no mundo dos bots de WhatsApp, é altamente recomendável que você siga o tutorial específico para a sua plataforma (Windows, Linux, Termux ou Pterodactyl).

Para aqueles que já têm alguma familiaridade, siga estes passos para uma configuração rápida:

  1. Prepare o Ambiente: Certifique-se de ter o Node.js (versão 18 ou superior) e o Git instalados.
  2. Baixe o Bot: Clone o repositório oficial do nazuna Bot do GitHub.
  3. Conecte ao WhatsApp: Utilize um número de telefone secundário e exclusivo para o bot.
  4. Explore os Comandos: Após a conexão bem-sucedida, envie /menu para o bot no WhatsApp para descobrir todas as funcionalidades.

💡 Dica para Iniciantes: Não se sinta sobrecarregado se os termos parecerem complexos. Cada seção deste documento foi cuidadosamente elaborada para fornecer instruções claras e passo a passo, tornando o processo acessível a todos.

🔌 Conectando o Bot ao WhatsApp

O nazuna Bot aproveita o modo multi-dispositivos do WhatsApp, o que significa que, após a configuração inicial, seu celular não precisa permanecer conectado à internet para que o bot funcione.

1. Conexão via QR Code (Recomendado)

Ao executar npm start, o terminal exibirá um QR code. No seu smartphone, abra o WhatsApp, vá em Configurações > Aparelhos Conectados, toque em Conectar um Aparelho e utilize a câmera para escanear o QR code no terminal.

2. Conexão via Código de Pareamento

Este método é útil para dispositivos sem câmera ou acesso remoto. Ao rodar npm start, o bot solicitará seu número. Após digitá-lo, o terminal fornecerá um código alfanumérico. No WhatsApp, vá em Configurações > Aparelhos Conectados, toque em Conectar com Número de Telefone e insira o código.

💻 Tutoriais de Instalação por Plataforma

O nazuna Bot é incrivelmente versátil e pode ser executado em diversas plataformas. Escolha o tutorial que corresponde ao seu ambiente para um guia de instalação detalhado.

🖥️ Windows: Guia Completo

Pré-requisitos: Instale Node.js e Git para Windows. Verifique a instalação com node -v e git --version.

Instalação: Abra o Prompt de Comando e execute os comandos para clonar o repositório e instalar as dependências:

git clone https://github.com/hiudyy/nazuna.git
cd nazuna
npm run config
npm run config:install

Executar: Para iniciar o bot, use npm start. Para mantê-lo online 24/7, você pode usar o NSSM (Non-Sucking Service Manager).

🐧 Linux: Instalação e Configuração

Pré-requisitos: Abra o terminal e instale Node.js e Git:

sudo apt update && sudo apt upgrade -y
sudo apt install -y nodejs git

Instalação: Clone o repositório e instale as dependências:

git clone https://github.com/hiudyy/nazuna.git
cd nazuna
npm run config
npm run config:install

Executar: Para iniciar o bot, use npm start. Para rodar em segundo plano, recomendamos usar o pm2 com npm install -g pm2 e pm2 start npm --name "nazuna" -- start.

📱 Termux (Android): Passo a Passo Detalhado

Instalar Termux: Baixe apenas do F-Droid. A versão da Google Play está desatualizada.

Configuração Inicial: Abra o Termux e execute:

pkg update && pkg upgrade -y
termux-setup-storage

Instalar Pré-requisitos: Instale as ferramentas necessárias:

pkg install nodejs git -y

Instalação do Bot: Clone o repositório para o armazenamento interno:

cd ~/storage/shared
git clone https://github.com/hiudyy/nazuna.git
cd nazuna
npm run config
npm run config:install

Executar: Inicie o bot com npm start. Mantenha o Termux rodando e desative as otimizações de bateria para o app.

☁️ Pterodactyl: Configuração em Servidores

Opção 1 (Recomendado): Use o Egg oficial do nazuna, que automatiza a maior parte do processo. Baixe o JSON do Egg em https://gitlab.com/hiudyy/nazuna-funcs/nazuna-egg.json, importe-o no painel de administração e crie um servidor usando este Egg.

Opção 2 (Manual): Crie um servidor com um Egg Node.js comum e execute os comandos de instalação manualmente via console, seguindo os mesmos passos das outras plataformas.

❓ Resolvendo Problemas Comuns (FAQ)

Problema Comum Solução Detalhada
"Comando não encontrado" (git, node, npm) Verifique se o Node.js e o Git estão instalados corretamente e no PATH do sistema.
QR Code não funciona ou expira rapidamente Verifique sua internet, aumente a janela do terminal ou tente o método de conexão com código de pareamento.
Bot desconecta frequentemente Verifique a estabilidade da sua internet e a quantidade de recursos (RAM/CPU) alocados para o bot.
Erro durante a instalação de dependências Tente limpar o cache do npm com npm cache clean --force e reinstale com npm install.
Termux para de funcionar em segundo plano Desative a otimização de bateria para o Termux nas configurações do Android.
